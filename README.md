<h1>🐤 FlappyBird </h1>

<h2>♟ Resumo </h2>
Criação de um app que se assemelha ao jogo original FlappyBird, usando a biblioteca, lib GDX, para Android em java, pude aprender a consumir a biblioteca e gerar animações em 2D.

<h2>🔥 Technologies</h2>
<ul>
<li>Java</li>
<li>lib GDX</li>
<li>Android</li>
<li>Android Studio</li>
</ul>
